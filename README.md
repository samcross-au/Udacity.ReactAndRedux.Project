# Udacity.React.WorkPolls

## About WorkPolls Project

This is a react project and it is part of Udacity React Nanodegree, specifically in the React Fundamentals course. This is an app that handles user-created polls within a workplace. You can create new log in to your account, view all polls (new and ones you have alreay answered) and vote on new polls.

## How to start 

Clone the repository, change directories, and use NPM to install the dependencies.

- $ git clone https://github.com/samcross-au/Udacity.ReactAndRedux.Project.git
- Navigate to the folder (e.g. `cd Udacity.ReactAndRedux.Project`)
- Install dependencies with `npm install`
- Start with `npm start`

The project can be viewed in the browser at
- [http://localhost:3000](http://localhost:3000)
